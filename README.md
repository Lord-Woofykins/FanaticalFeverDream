# Fanatical Fever Dream
This is my IST assessment game for Term 3 2025.

## How To Play
Move player -- WASD OR Arrow Keys
Attack -- J OR Z
Pause Menu + Save Game -- ESCAPE
Exit Game -- ESCAPE x 2 (or once from pause menu)

## Custom Features
1. Crouching (Custom Manoeuvre)
2. Saving the game
3. Custom keybinds
4. Pause menu
